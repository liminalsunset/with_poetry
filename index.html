Getting started in Django 4 utilizing poetry (a packaging and dependency manager).

This is currently still in progress and incomplete.


  1. Check your Python version (you need at least 3.8 to use Django 4, and 3.7+ to use poetry):

    python3 --version

  2. Install poetry:

    curl -sSL https://install.python-poetry.org | python3 -

    more installation options in the docs
  3. Make a directory for the project and cd into it.
  4. Init:

    poetry init
